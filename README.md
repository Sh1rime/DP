# PDF Analyzer Microservice

Этот проект позволяет загружать PDF-документы и получать их анализ в виде структурированного JSON, используя стек FastAPI + Celery + Redis + Tesseract. 

## Запуск через Docker

1. Убедитесь, что установлены **Docker** и **docker-compose**.
2. Склонируйте репозиторий (или скопируйте файлы проекта) на локальную машину.
3. Перейдите в директорию проекта и выполните сборку и запуск контейнеров:
   ```bash
   docker-compose up --build
